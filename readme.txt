Show the edit status of a post in the front-end.

== Description ==
This plugin adds a "*user* is currently editing" text to the front-end single post when someone is editing the post.

== Installation ==
I recommend to install plugins via your WordPress Dashboard.

For manual installation:
Download mklasen's Post Edit Status Plugin (http://wordpress.org/extend/plugins/mklasens-thumbnail-slider).

To install mklasen's Post Edit Status follow these steps:
- Upload the folder \"mklasens-post-edit-status\" to your WordPress plugin folder (/wp-content/plugins/)
- Activate the plugin through the 'Plugins' menu in WordPress
